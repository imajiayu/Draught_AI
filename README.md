# Draught_AI
A simple draught AI engine based on C/C++ and Qt5
# Command
* START [SIDE]

  SIDE:O as black and 1 as white
* TURN

  AI engine execute one step
* PLACE [step nums] [x0] [y0] [x1] [y1]......

  execute one step
* END

  end the game
